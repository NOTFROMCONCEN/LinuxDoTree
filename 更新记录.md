# 更新记录

> 本文档汇总最近几轮针对 `linuxdotree` 的功能与架构优化，便于回顾变更背景与影响范围。

## 1.0.1-beta.1（初始版本）
- 初始化扩展工程，提供 Linux.do 帖子页树形评论跳转的核心能力。
- 建立跨浏览器构建与发布基础结构（Chrome / Edge / Firefox）。

对应提交：`07856ae`

---

## 1.1.0（更新检测能力）
### 主要更新
- 在弹窗中加入版本更新检测能力，支持检测 GitHub 最新 release。
- 增加缓存策略，避免每次打开弹窗都发起网络请求。
- 补充扩展权限以支持更新检测所需的外部请求。

### 影响文件（核心）
- `src/common/popup.js`
- `src/manifests/chrome.json`
- `src/manifests/edge.json`
- `src/manifests/firefox.json`

对应提交：`f4caf1b`

---

## 1.2.0（共享设置模块 + 路由/DOM 监听优化）
### 主要更新
- 新增共享设置模块 `src/common/settings.js`，统一管理：
  - `DEFAULT_SETTINGS`
  - `FORCED_DISABLED_FIELDS`
  - `normalizeSortMode`
  - `normalizeSettings`
- `content.js`、`popup.js`、`options.js` 改为复用共享设置逻辑，减少重复代码与配置漂移。
- `settings.js` 注入到：
  - 内容脚本加载链（manifest）
  - popup/options 页面脚本加载链
- 路由与页面刷新机制优化：
  - 统一路由变化处理
  - 取消低效轮询逻辑
  - MutationObserver 改为更聚焦区域并过滤关键变更触发

### 影响文件（核心）
- `src/common/settings.js`
- `src/common/content.js`
- `src/common/popup.js`
- `src/common/options.js`
- `src/common/popup.html`
- `src/common/options.html`
- `src/manifests/chrome.json`
- `src/manifests/edge.json`
- `src/manifests/firefox.json`

对应提交：`7dbf47d`

## 1.2.1（第三轮细化优化）
### 主要更新
- `content.js`：History API 包装标记由字符串升级为 `Symbol`，降低属性名冲突风险。
- `popup.js`：升级版本比较逻辑，增强对语义化版本与预发布标识（如 `beta`）的比较可靠性。
- `popup.js`：更新提示横幅点击事件改为覆盖式绑定，避免重复绑定风险。

### 影响文件（核心）
- `src/common/content.js`
- `src/common/popup.js`

对应提交：`b9c3f3f`
